# 🍽 Simply Homemade

Simply Homemade is a recipe-sharing web application developed for C237 Software Application Development CA2.

Users can create, view, edit and delete their own recipes. They can also favourite, rate and comment on recipes.

## Technologies

- Node.js
- Express
- EJS
- MySQL
- Azure Database for MySQL
- bcrypt
- express-session
- Multer
